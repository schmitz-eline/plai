# Projet client - PLAI

Ceci est un projet client réalisé dans le cadre du cours de Design Web en 2ème année d’infographie à la HEPL.

Le PLAI (Pôle Liégeois d’Accompagnement vers une école Inclusive) accompagne les élèves à besoins spécifiques et les équipes éducatives dans la mise en place d’aménagements raisonnables.
